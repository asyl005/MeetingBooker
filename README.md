# MeetingBooker

MeetingBooker – мектеп мұғалімдері мен ата-аналар арасындағы кездесулерді тиімді жоспарлауға және өзара байланысты жеңілдетуге арналған iOS қосымшасы. Жоба мектеп әкімшілігі мен ата-аналар қауымдастығының уақытын үнемдеу мақсатында әзірленген.

## Жобаның сипаттамасы

Қосымша екі түрлі рөлдегі қолданушыларға арналған: "Мұғалім" және "Ата-ана". Жүйе кездесу уақытын белгілеу, сұраныстарды растау немесе қабылдамау, сондай-ақ тараптар арасында тікелей хабарлама алмасу мүмкіндіктерін ұсынады. Деректер жергілікті жадта сақталады, бұл қосымшаны интернетсіз де (ішінара) және серверсіз тестілеуге мүмкіндік береді.

## Функционалдық мүмкіндіктер

### Ата-аналар үшін:

  * Мұғалімдер тізімін және олардың пәндерін қарау.
  * Мұғалімнің бос уақыт кестесіне сәйкес кездесуге сұраныс жіберу.
  * Жіберілген сұраныстардың мәртебесін бақылау (Күтуде, Бекітілді, Бас тартылды).
  * Өз балаларының тізімін басқару.
  * Мұғаліммен жеке чат арқылы сөйлесу.

### Мұғалімдер үшін:

  * Келіп түскен кездесу сұраныстарын қарау.
  * Сұраныстарды қабылдау немесе олардан бас тарту.
  * Бекітілген кездесулердің тізімін көру.
  * Ата-аналармен чат арқылы байланысу.

### Жалпы мүмкіндіктер:

  * **Авторизация және Тіркелу:** Қауіпсіз кіру және жаңа қолданушыларды тіркеу жүйесі.
  * **Таймер:** Кездесу басталғанға дейінгі қалған уақытты көрсететін кері санақ жүйесі.
  * **Чат:** Әр кездесу үшін оқшауланған хабарлама алмасу тарихы.
  * **Интерфейс:** Ашық көк түсті палитраға негізделген таза және интуитивті UI.

## Техникалық стек

Жоба iOS платформасына арналып, заманауи Apple технологияларын қолдана отырып жазылған:

  * **Тіл:** Swift 5
  * **UI Framework:** SwiftUI
  * **Архитектура:** MVVM (Model-View-ViewModel)
  * **Реактивті бағдарламалау:** Combine
  * **Деректерді сақтау:** UserDefaults (Local Persistence), Codable протоколы
  * **Минималды iOS нұсқасы:** iOS 15.0+

## Жоба құрылымы

Код архитектуралық тазалықты сақтау үшін келесідей құрылымдалған:

  * `Models` – Деректер құрылымы (User, Meeting, Chat, Teacher).
  * `ViewModels` – Бизнес-логика және UI жаңартулары (AuthViewModel, MeetingViewModel, ChatViewModel).
  * `Views` – Пайдаланушы интерфейсінің экрандары.
  * `Helpers` – Көмекші құралдар мен сервистер (Database, CountdownTimer).

## Орнату және іске қосу

Жобаны жергілікті компьютерде іске қосу үшін келесі қадамдарды орындаңыз:

1.  **Репозиторийді жүктеу:**
    Терминал арқылы жобаны клондаңыз немесе ZIP архивін жүктеп алыңыз.

    ```bash
    git clone [репозиторий сілтемесі]
    ```

2.  **Жобаны ашу:**
    `MeetingSchedulerApp.xcodeproj` файлын Xcode ортасында ашыңыз.

3.  **Баптаулар:**
    Signing & Capabilities бөлімінде Team параметрін өз Apple ID аккаунтыңызға ауыстырыңыз (егер құрылғыға орнатқыңыз келсе).

4.  **Іске қосу:**
    Симуляторды таңдап (мысалы, iPhone 15), `Cmd + R` батырмасын басыңыз немесе "Run" белгішесін шертіңіз.
